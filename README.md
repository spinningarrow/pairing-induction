# Roman Numerals Kata
Convert Roman numerals into decimal notation.

## Conversion table

| Roman Numeral | Decimal Value |
|---------------|---------------|
| I             | 1             |
| IV            | 4             |
| V             | 5             |
| IX            | 9             |
| X             | 10            |
| XL            | 40            |
| L             | 50            |
| XC            | 90            |
| C             | 100           |
| CD            | 400           |
| D             | 500           |
| CM            | 900           |
| M             | 1000          |

## Examples

```
I -> 1
II -> 2
IV -> 4
V -> 5
CCIII -> 203
MCMXCII - 1992
```
